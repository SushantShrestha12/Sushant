Python Turtle

Introduction

In this tutorial we are gonna learn some of the basic python program.
To draw, Python turtle provides many functions and methods i.e. forward, backward, penup, pendown etc.

Circle Pattern:-
            import turtle
            turtle.Screen() 
            turtle.bgcolor('black') 
            t=turtle.Turtle() 
            t.speed(10) 
            t.pencolor('green') 
            t.fillcolor('red') 
            t.begin_fill() 
            for i in range(36): 
                t.circle(100) 
                t.left(10) 
            t.end_fill() 
            turtle.done()

Flower:-
              import turtle

              turtle.Screen()
              t=turtle.Turtle()
              t.speed(10)
              
              t.color('red')
              t.begin_fill()
              t.fillcolor('yellow')
              for i in range (24):
                  t.forward(250)
                  t.left(165)
              t.end_fill()
              
              t.color('green')
              t.begin_fill()
              t.fillcolor('red')
              t.left(180)
              for i in range (24):
                  t.forward(250)
                  t.right(165)
              t.end_fill()
              
              t.color('blue')
              t.begin_fill()
              t.fillcolor('pink')
              t.left(90)
              t.forward(300)
              for i in range (23):
                  t.right(165)
                  t.forward(250)
              t.end_fill()
              
              t.color('purple')
              t.begin_fill()
              t.fillcolor('black')
              t.left(20)
              t.penup()
              t.forward(360)
              t.pendown()
              for i in range (24):
                  t.right(165)
                  t.forward(250)
              t.end_fill()
              
              turtle.done()
